# Boursati Marketing

Start with:

`BOURSATI_FINAL_SYSTEM/`

That folder contains the approved final design system, brand guidelines, marketing plan, captions, calendar, launch assets, platform assets, contact sheets, source files, scripts, tests, and captures.

Everything else is archived history.

## Current Folder Map

- `BOURSATI_FINAL_SYSTEM/` - active source of truth.
- `ARCHIVE/` - old experiments, rejected directions, older exports, Higgsfield work, previous renderers, and reference repos.
- `.superpowers/` - hidden local planning metadata.

## Review Path

1. Open `BOURSATI_FINAL_SYSTEM/04_assets/launch_batch/`.
2. Use `BOURSATI_FINAL_SYSTEM/04_assets/contact_sheets/` for fast visual review.
3. Review captions in `BOURSATI_FINAL_SYSTEM/03_content_strategy/captions/`.
4. Use `BOURSATI_FINAL_SYSTEM/04_assets/platform_static/` for the broader posting bank.

## Working Rule

Future Boursati marketing work should happen inside `BOURSATI_FINAL_SYSTEM/`.

Do not use archived material unless we intentionally decide to revive something.
