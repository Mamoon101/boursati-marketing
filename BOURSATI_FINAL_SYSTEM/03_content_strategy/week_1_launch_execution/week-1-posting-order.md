# Week 1 Posting Order

## Principle

Week 1 should establish the account clearly before deeper education begins. Start with the Instagram grid, then move into feed proof, stories, and X banners.

## Day 0 - Account Setup

Before posting:

1. Set profile name and bio from `instagram-profile-launch-setup.md`.
2. Add App Store link or `boursati.app` in bio.
3. Prepare all nine Instagram grid images.
4. Keep Week 2 and Week 3 parked.
