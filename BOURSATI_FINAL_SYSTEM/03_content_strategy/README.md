# Content Strategy

Use this folder for planning what to post and how to caption it.

- `captions/` - final launch captions and caption banks.
- `calendar/` - posting sequence and rotation calendar.
- `posting_plan/` - practical posting order.
- `content_maps/` - asset maps, carousel plans, and review tables.
- `week_1_launch_execution/` - active Week 1 posting order, captions, checklist, and Instagram profile setup.
