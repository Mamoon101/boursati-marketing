# Boursati Final System

Start here. This folder contains the approved Boursati brand direction, marketing plan, production assets, and source system.

## Where To Look

- `../About Boursati.md` - product context for agents before creating strategy, copy, assets, captions, videos, or automation.
- `01_brand_guidelines/` - approved art direction and brand rules.
- `02_marketing_plan/` - launch positioning and channel strategy.
- `03_content_strategy/` - captions, calendar, posting plan, and content maps.
- `04_assets/ready_to_post_by_week/` - active approved posting batches.
- `04_assets/future_batches_parked/` - Week 2 and Week 3 parked for later.
- `04_assets/launch_batch/` - earlier curated first batch, kept as reference until the weekly structure is approved.
- `04_assets/_legacy_ready_by_platform/` - original platform-organized package, kept for traceability.
- `04_assets/contact_sheets/` - quick visual review sheets.
- `05_source_system/` - content JSON, renderer scripts, templates, tests, assets, and captures used to regenerate the system.
- `06_references/approved_design_direction/` - reference material for the approved visual direction.
- `07_publishing_automation/` - Postiz-based automation queue and scheduling scripts.

## Review Order

1. Use `04_assets/ready_to_post_by_week/dark_daily_batch_v1/` for the current two-week dark-only daily run.
2. Use `contact-sheet-instagram-grid-preview.png` before posting Instagram assets.
3. Check captions in `04_assets/ready_to_post_by_week/dark_daily_batch_v1/captions.md`.
4. Use older launch assets only when explicitly needed.
5. Do not revive parked or legacy batches without review.

## Automation

Week 1 publishing automation lives in `07_publishing_automation/`.

Live posting requires a launch date and connected Postiz integrations for Instagram/X.

## Rule

Anything outside this folder is archive or history unless intentionally brought back.
